## Description

A Discord bot for servers with collaborative playlists.
Secondary function: a bot for bands.

Comands:

/recent
/features
/analysis
/artists
/playlistfeatures
/recommendation
/lyrics
/subscribe

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
